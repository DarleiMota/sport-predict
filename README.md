# 📊 SportPredict

API backend desenvolvida em Java com Spring Boot para análise estatística de partidas de futebol, utilizando distribuição de Poisson para estimativa de eventos como gols.

---

## 🚀 Objetivo

Construir uma API capaz de analisar dados de partidas e gerar previsões estatísticas com base em modelos matemáticos aplicados ao futebol.

---

## 🛠️ Tecnologias utilizadas

- Java  
- Spring Boot  
- PostgreSQL  
- JPA / Hibernate  
- REST API  

---

## 🧠 Estrutura do projeto

O projeto segue arquitetura em camadas:

- **Controller:** responsável por receber as requisições HTTP  
- **Service:** onde estão as regras de negócio  
- **Repository:** responsável pela comunicação com o banco de dados  

---

## ⚙️ Funcionalidades

- Cadastro e gerenciamento de equipes  
- Integração com API externa de dados esportivos  
- Processamento estatístico utilizando distribuição de Poisson
